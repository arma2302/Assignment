6-What is Header and Content Components in React Js?
-Header Component: A Header component is a reusable piece of code that represents the top section of a web page. It typically contains the website's logo, navigation menu, and other essential elements that are consistent across all pages. The Header component is usually placed at the top of the page and remains fixed, even when the user scrolls down.
-Content Component: A Content component, on the other hand, represents the main content area of a web page. It contains the dynamic content that changes depending on the page or user interaction. The Content component is usually placed below the Header component
